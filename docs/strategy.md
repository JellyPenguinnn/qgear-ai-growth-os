# Strategy

Q-GEAR AI Growth OS is designed for a Singapore/Malaysia-based individual investor starting with about USD 10,000 and researching US-listed AI-era growth companies.

The strategy is fundamental-first. Technicals are used for timing and risk confirmation only.

## Defaults

| Setting | Default |
|---|---:|
| Base CAGR objective | 18-22% |
| Strong objective | 22-25% |
| Stretch objective | 25-30% |
| Normal max drawdown budget | 25-30% |
| Hard drawdown limit | 35% |
| Live positions | 6-10 |
| Cash buffer | 10-20% |
| Max single stock | 15% |
| Holding period | 1-5 years |
| Benchmarks | SPY, QQQ, XLK, SMH |

These are research assumptions, not return promises.

## Position Sizing

| Status | Allocation | USD amount on $10k |
|---|---:|---:|
| Research only | 0% | $0 |
| Starter | 2.5-5% | $250-$500 |
| Normal | 5-8% | $500-$800 |
| High conviction | 8-12% | $800-$1,200 |
| Exceptional | 12-15% | $1,200-$1,500 |
| Absolute cap | 15% | $1,500 |

No action is justified unless evidence, thesis, valuation, technical regime, risk budget, and evidence freshness all clear.
